## Проект 'Игры разума'
Этот проект включает в себя пять игр:
- 'Проверка на четность'
- 'Калькулятор'
- 'Наибольший общий делитель'
- 'Арифметическая прогрессия'
- 'Простое ли число?'
### Минимальные требования:
- Python 3.6 и выше
- pip (версия 19 и выше)
- poetry 1.2.0 и выше
### Инструкции по установке проекта:
```python
make install
make test
make lint
make selfcheck
make check
make build
```
### Инструкции по запуску игр:
```python
brain-calc
brain-even
brain-gcd
brain-prime
brain-progression
```
